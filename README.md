**TrashWise** adalah aplikasi berbasis CLI (*Command Line Interface*) yang dirancang untuk meningkatkan kesadaran masyarakat tentang pengelolaan sampah. Aplikasi ini tidak hanya memberikan edukasi, tetapi juga menyediakan alat praktis untuk mengonversi sampah menjadi uang, menemukan lokasi bank sampah, dan berdonasi untuk lingkungan.

TrashWise hadir sebagai solusi digital sederhana untuk membantu pengguna memahami konsep **3R (Reuse, Reduce, Recycle)**. Aplikasi ini berfokus pada wilayah **Yogyakarta dan sekitarnya**, membantu menghubungkan masyarakat dengan bank sampah terdekat dan memberikan estimasi nilai ekonomi dari sampah anorganik yang mereka miliki.
